# Einkaufslisten Rechner

Ein einfaches Python-Kommandozeilenprogramm zum Erstellen von Einkaufslisten mit automatischer Preisberechnung.

## Features

- Produkte mit Preisen hinzufügen
- Automatische Summenberechnung
- Übersichtliche Ausgabe der Einkaufsliste

## Verwendung

```bash
python Einkaufslisten_Rechner.py
```

Das Programm fragt nach Produktnamen und Preisen. Gib `fertig` ein, um die Eingabe zu beenden und die Gesamtsumme zu sehen.

## Beispiel

```
Bitte Produkt eingeben: Milch
Bitte Preis eingeben: 1.29
Bitte Produkt eingeben: Brot
Bitte Preis eingeben: 2.49
Bitte Produkt eingeben: fertig

=== EINKAUFSLISTE ===
Milch: 1.29 €
Brot: 2.49 €
------------------------------
Summe: 3.78 €
```

## Technologien

- Python 3
- Dictionaries für Datenspeicherung

## Lernziele

- Arbeiten mit Dictionaries
- While-Loops und Benutzereingaben
- Try/Except für Fehlerbehandlung
- Iteration über Dictionary-Items
